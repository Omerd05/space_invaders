A miniature project used for studying Kotlin syntax and its basic concepts. 

Short description:
In terminal semi space invaders game. 

To run:
pull the repo, make sure you have java 17 installed
run:

./gradlew tasks
./gradlew run

Some notes:
  Eventually I used lanterna for terminal gui, although originally I tried to use ANSI escape sequences but it didn't work smoothly. 

As for now, the program supports basic input handling (moving player's avatar and shooting) and automated moving of the invaders. 
I consider the design for now as suboptimal, maybe later on the feature I will remodel it (probably not).
